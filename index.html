PROJECTS

1. Local Shop Marketing

The Local Shop Marketing App is a full-stack application designed to empower small, local businesses by providing them with a digital platform to showcase their products and services. The app bridges the gap between local shops and customers, enabling shop owners to set up profiles, upload product catalogs, create promotional offers, and interact with customers via an integrated messaging system.

For end users, the app offers a user-friendly interface that includes features such as product browsing, cart management, order tracking, and location-based search functionality.

The frontend is developed using HTML, CSS, and Bootstrap, ensuring a responsive and intuitive user interface. The backend logic is implemented in Python with MySQL as the database, all managed via JetBrains PyCharm. Additionally, a native mobile application was built using Android Studio and Java to extend functionality on mobile devices.

2. College Enquiry Chatbot

This project develops a chatbot to streamline college communications between students, staff, and administration. Through automation, the system will minimize human intervention, allowing staff to focus on more critical tasks while students receive accurate and timely information.

This project falls within the domains of artificial intelligence, chatbot development, and education technology. It involves the implementation of machine learning and NLP techniques through tools such as Dialogflow, as well as web development framework Django for backend support.
